# Technique Catalogue

Available techniques from the Pixels2GenAI curriculum, organized by visual potential for static exhibition prints. Each entry notes what it produces visually and what it combines well with.

## Module 1: Pixel Fundamentals

| Technique | Visual Character | Combines With |
|-----------|-----------------|---------------|
| RGB manipulation | Precise color control, gradients, channel isolation | Everything -- this is the base layer |
| Color space conversion (HSV, LAB) | Perceptual color harmonies, smooth hue rotations | Noise fields, activation landscapes |
| Cellular automata (Game of Life) | Binary emergence patterns, textile-like grids | Fractal zoom, color mapping, noise seeding |
| Reaction-diffusion (Gray-Scott) | Organic spots/stripes, coral/animal-like textures | Activation functions, distance fields |
| Truchet tiles | Infinite maze patterns, flow-like tessellations | Perlin noise for rotation selection, color gradients |
| Wang tiles | Non-repeating seamless patterns | Large-format prints, texture synthesis |

## High-Impact Combinations (Unexplored)

These pairings have strong visual potential and haven't been done in the exhibition yet:

1. **Reaction-diffusion + Activation functions**: Turing patterns shaped by ReLU/sigmoid thresholds
2. **Strange attractors + Custom color mapping**: Lorenz/Rossler trajectories as luminous traces
3. **Domain-warped noise + Voronoi**: Organic cell structures with fluid distortion
4. **L-systems + Perlin wind + Seasonal color**: Forest canopy viewed from above
5. **DLA crystal growth + AI fabric patterns**: Crystalline structures filled with learned textures
6. **Mandelbrot deep zoom + African fabric palette**: Fractal geology in cultural colors
7. **Wave interference + SDF shapes**: Standing waves constrained by mathematical boundaries
8. **Differential growth + Convolution edge detection**: Organic forms with highlighted structure
9. **Multiple strange attractors overlaid**: Chaotic systems in dialogue, color-coded
10. **Fourier decomposition of AI-generated patterns**: Frequency analysis of what GANs learn
